Act as an expert UI/UX and Front-End Developer. Write clean, semantic, and fully responsive HTML5 and CSS3 code for a modern User Profile (Avatar) Website.

### Website Structure & Requirements:
1. **Top Header/Navigation Bar:**
   - Website logo/name placeholder on the left.
   - Navigation links (Dashboard, My Orders, Support) on the right.

2. **Hero Profile Section (Main Focus):**
   - A distinct background area (maybe with a subtle texture or gradient).

4. **Footer:**
   - Simple footer with copyright info and social media icons.

### Technical Guidelines:
- Use semantic HTML5 tags exclusively.
- Use CSS Flexbox and Grid for responsive layouts (ensure it works perfectly on mobile devices).

- Pure HTML and CSS only—no external CSS frameworks. Use clear code comments.
